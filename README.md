# Estoque

Backend de controle de estoque desenvolvido em Java com Quarkus,
focado em organização de código,
boas práticas de arquitetura e APIs REST performáticas.

### Tecnologias usadas
* Java
* Quarkus
* REST API
* Hibernate ORM com Panache
* MySql
* Bean validation
* Open API / Swagger

### Como rodar o projeto

```bash
/gradlew quarkusDev
```

A aplicação sobe em:<br>
API: http://localhost:8080<br>
